# Card Memory Game

An interactive memory game built with React, TypeScript, and Vite.

## How to run it locally

Follow these steps from your terminal, making sure you are in the project's root folder:

### 1. Install dependencies
Download and install all the necessary packages by running:
```bash
npm install
```

### 2. Start the development server
Start the application in your local environment with:
```bash
npm run dev
```
*Once finished, the terminal will show you a link (usually `http://localhost:5173/`). Click on it or copy it into your browser to see the game running.*
